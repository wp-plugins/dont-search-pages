Exclude static pages in internal blog search.


== Description ==
Execute blog search excluding pages. Fast, simple and effective.


= Features =
* Very fast execution
* No user interface required: Install, done!
* Tiny search query in only one sql statement

== Installation ==
1. Download *Don't Search Pages* plugin
1. Unzip the archive
1. Upload the folder *dont-search-pages* into *../wp-content/plugins/*
1. Go to tab *Plugins*
1. Activate *Don't Search Pages*
1. Ready
